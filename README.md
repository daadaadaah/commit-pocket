## 🚀 소개
- Git Commit Pocket 은 Git `[Commit ID]` 를 보관할 수 있는 CLI 입니다.
- Git 명령어(예 : rebase, cherry-pick 등)을 사용할 때, 필요한 `[Commit ID]` 를 보관하기에 유용한 CLI 입니다.

## 🚀 사용법
### 🔥 설치

```bash
$ npm install -g commit-pocket
```

### 🤹 시작하기

```bash
$ commit-pocket
```
